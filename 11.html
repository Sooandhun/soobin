Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server of from local storage and render the, into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the apperance of the documents

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, maybe embedded into the rendered page. It provides into the rendred page. It provies a means to create structured documents by denoting structural semantics for text such as heading, paragraphs, lists, links, quotes and other items.
